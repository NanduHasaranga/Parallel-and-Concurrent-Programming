SENATE BUS STOP SIMULATION
=========================

HOW TO COMPILE:
1. Open terminal in the project directory
2. Run: javac SenatesBus.java

HOW TO RUN:
1. After compilation, run: java SenatesBus
2. Output will show rider arrivals, bus arrivals, boarding events, and departures

HOW TO STOP:
1. Press Ctrl + C in the terminal to terminate the simulation
